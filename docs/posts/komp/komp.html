Компьютерная терминология

Published

Вот некоторые компьютерные термины:

ОЗУ (RAM) — (Random Access Memory, с англ. — «Запоминающее устройство с произвольным доступом») - оперативная память

ПЗУ (ROM)— Постоя́нное запомина́ющее устро́йство — энергонезависимая память, используется для хранения массива неизменяемых данных - постоянная память

IP-адрес (IP address - (от англ. Internet Protocol) — уникальный числовой идентификатор устройства в компьютерной сети, работающей по протоколу IP

Программирование - programming [ˈprəʊgræmɪŋ] — человек даёт компьютеру алгоритм действий.

3D-печать - 3D printing — это создание трехмерного объекта по модели САПР или цифровой 3D-модели. Это можно сделать с помощью различных процессов, в ходе которых материал наносится, соединяется или затвердевает под управлением компьютера, при этом материал добавляется слой за слоем (например, пластмассы, жидкости или порошкообразные материалы)

Интернет вещей - (internet of things, IoT) — устройства, которые подключаются к интернету и могут взаимодействовать друг с другом.

Кеш - cache [kæʃ] — область программы, позволяющая получить быстрый доступ к информации, которую пользователь недавно использовал.

Cookie [ˈkʊkɪ] — куки - маленькие файлы, которые сайт отправляет автоматически на компьютер пользователя.

Облако (Cloud- remote server)— удалённый сервер, на котором хранятся данные пользователя.
